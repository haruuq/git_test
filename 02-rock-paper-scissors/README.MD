# Rock, Paper, Scissors (Console Version)

A simple **Rock, Paper, Scissors** game built with **JavaScript** as part of my learning journey through **The Odin Project**.

> **Note:** This project does **not** include a graphical user interface (GUI). It is designed to be played **entirely through the browser console and prompt dialogs**.

## About the Project

The goal of this project was to practice and reinforce core JavaScript concepts rather than create a polished user interface. The game runs for five rounds, compares the player's choice against a randomly generated computer choice, and keeps track of the score until a winner is determined.

## What I Practiced

While building this project, I worked with:

* Variables and constants (`let` and `const`)
* Functions and arrow functions
* Conditional statements (`if`, `else if`, `else`)
* Logical operators (`&&` and `||`)
* Loops
* Random number generation
* User input with `prompt()`
* Console output with `console.log()`
* Updating and tracking scores with program state

## How to Play

1. Open the project in your browser.
2. Open the browser's Developer Tools and navigate to the **Console**.
3. Run the game (or refresh the page if it starts automatically).
4. Enter your choice when prompted (`Rock`, `Paper`, or `Scissors`).
5. The game will play five rounds and display the results and final score in the console.
